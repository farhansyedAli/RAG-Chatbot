
🚀 RAG Chatbot
==========================

✨ Features
------------

The following are some of the key features of the Document Similarity Analyzer project:

1. **Document Upload**: Users can upload documents in various formats, including PDF, Word, and text files.
2. **Document Processing**: The project uses PyMuPDF and docx libraries to process and extract text from uploaded documents.
3. **Sentence Embeddings**: The project uses the sentence-transformers library to compute sentence embeddings, which are then used to Give Answers from tese file.
4. **Interactive Interface**: The project uses the Gradio library to create a simple and interactive web interface, allowing users to upload and Answer the questions from documents easily.
5. **Support for Multiple Document Formats**: The project supports multiple document formats, including PDF, Word, and text files.
6. **Easy to Use**: The project provides a simple and intuitive interface, making it easy for users to upload and get answers from documents without requiring extensive technical knowledge.

📁 Project Structure
--------------------

The project consists of the following folders and files:

* **app.py**: The main application file, containing the code for the document similarity analyzer.
* **requirements.txt**: The file containing the dependencies required to run the project.
* **docs**: The folder containing documentation and guides for the project.
* **models**: The folder containing pre-trained models used in the project.
* **data**: The folder containing sample data used for testing and demonstration purposes.

⚙️ How to Run
--------------

To run the project, follow these steps:

1. **Setup**: Install the required dependencies by running `pip install -r requirements.txt` in the terminal.
2. **Environment**: Create a new environment using `python -m venv env` and activate it using `source env/bin/activate`.
3. **Build**: Build the project by running `python app.py` in the terminal.
4. **Deploy**: Deploy the project by running `gradio app.py` in the terminal, which will start the Gradio server and make the application available at `http://localhost:7860`.
